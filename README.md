# ERP Vendor Management Microservice

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

## Features
- Add Vendor
- Get All Vendors

## API Endpoints
- POST /vendors
- GET /vendors

## Database
PostgreSQL connected using JPA & Hibernate

## How to Run
- Clone repo
- Configure DB in application.properties
- Run VendorApplication.java
- Access: http://localhost:8080/vendors
